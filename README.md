# Trovn Client

This is the client-side application for Trovn, a web platform for renting rooms and finding roommates.

## Features

- **User Authentication:** Sign up, sign in, and manage user profiles.
- **Listing Management:** Create, view, update, and delete room listings.
- **Search and Filtering:** Advanced search and filtering options to find the perfect room.
- **Interactive Map:** View listings on an interactive map.
- **Real-time Chat:** Communicate with other users in real-time.
- **Booking and Payments:** Book rooms and handle payments securely.
- **Admin Dashboard:** Manage users, listings, and other aspects of the platform.
- **AI-Powered Features:** Utilizes Google's Generative AI for an enhanced user experience.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Routing:** React Router
- **State Management:** Zustand
- **UI Components:** Shadcn UI, Ant Design
- **Maps:** Mapbox GL
- **Real-time Communication:** Socket.IO
- **Form Handling:** React Hook Form, Zod
- **AI:** Google Generative AI

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/cuong07/trovn-client.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

### Running the Application

To run the application in development mode, use the following command:

```sh
npm run dev
```

This will start the development server at `http://localhost:5173`.

## Available Scripts

In the project directory, you can run:

- `npm run dev`: Runs the app in the development mode.
- `npm run build`: Builds the app for production to the `dist` folder.
- `npm run lint`: Lints the code using ESLint.
- `npm run preview`: Serves the production build locally.
